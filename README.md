## About The Project

This is a learning exercise, focused on Operating Systems and what makes them work (or not!)

The primary focus of the project is to learn and master:
* Synchronization
* Memory Management
* Syscalls
* OS Testing and troubleshooting




### Built With

* C
* Linux
* GDB
